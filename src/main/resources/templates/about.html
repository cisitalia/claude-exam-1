기술 스택

Backend

  • • Java 21
  • • Spring Boot 3.3.5
  • • Spring Data JPA
  • • Thymeleaf
  • • MySQL 5.7
  • • Gradle (Kotlin DSL)

Frontend

  • • Tailwind CSS v4
  • • Alpine.js v3
  • • HTMX
  • • esbuild
  • • Pretendard 폰트

프로젝트 특징

  • • SSR (Server-Side Rendering): Thymeleaf를 통한 서버 사이드 렌더링
  • • 반응형 디자인: Tailwind CSS로 모바일 우선 반응형 UI
  • • 인터랙티브 UI: Alpine.js로 가벼운 클라이언트 사이드 기능
  • • HTMX 통합: 페이지 새로고침 없는 동적 콘텐츠 로딩
  • • 개발 효율성: Hot reload와 자동 빌드 지원
  • • 한국어 지원: Pretendard 폰트와 한국어 최적화
